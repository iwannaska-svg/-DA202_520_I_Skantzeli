# Midnight Dialogue
## Jazz Electroacoustic Composition

## Playback Instructions

1. Open the file:
Midnight_Dialogue_FINAL_WORKING.scd

2. Start SuperCollider Server:
Ctrl + B

3. Run Part 1:
Click inside Part 1 and press Ctrl + Enter.

4. Run:
Part 2 - Intro + Jazz Theme
Part 3 - Walking Bass
Part 8 - Electroacoustic Atmosphere

For each part press Ctrl + Enter.

5. Run FINAL PERFORMANCE:
Click inside FINAL PERFORMANCE and press Ctrl + Enter.

The composition will play automatically.
